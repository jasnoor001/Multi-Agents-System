Developed a Multi-Agent AI Research System using LangChain, LCEL (LangChain Expression Language), and an LLM. The project includes two custom tools: the first generates structured insights and background information on a given topic, while the second retrieves in-depth, reliable, and up-to-date news from trusted sources. Two specialized AI agents were designed, each integrated with the LLM to handle its respective task efficiently. Finally, an LCEL-based pipeline (pipeline.py) was implemented to orchestrate the interaction between all agents, tools, and components, creating a modular and scalable research workflow.
